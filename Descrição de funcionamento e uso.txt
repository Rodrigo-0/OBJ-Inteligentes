No projeto, pude realizar a implementação de um sistema de irrigação monitorada por um sensor de umidade do solo. Nele, é capaz de observar que o sensor consegue captar a informação de umidade e dar insumo para que um módulo relé ative uma válvula solenoide e as ações de irrigação possam ter sua devida objetificação.
O uso para reprodução, consiste em termos um Arduino UNO, uma protoboard, jumpers machos e femeas conforme o necessário, sensor de umidade, válvula solenoide e o rele. Tendo todos estes equipamentos, aqui no github tem o código utilizado em conjunto com o Blynk, aplicativo onde é possível configurar uma comunicação remota e executar os devidos comando direto do aplicativo.
